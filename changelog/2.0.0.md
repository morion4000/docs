# Version 2.0.0 (2026-03-12)

This is a major release with sweeping changes across the entire platform — both the UI and the backend have been significantly reworked.

## What's New

- Complete UI overhaul with a modern, redesigned interface
- Major backend architecture improvements for better performance and reliability
- **Email Campaigns** — create, manage, and send email campaigns directly from the platform
- **A/B Testing** — run experiments on your sites with built-in A/B testing support
- Numerous new features and enhancements across the board
