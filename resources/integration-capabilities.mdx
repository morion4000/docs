GetSite offers robust integration capabilities that allow you to connect your websites with various third-party services and platforms. These integrations extend the functionality of your GetSite websites, enabling seamless data flow between systems and enhancing your overall web presence.

## Analytics Platforms

Track website performance and visitor behavior with analytics integrations:

### Supported Analytics Tools

- **Google Analytics**: Comprehensive website analytics
- **Google Search Console**: Monitor search engine performance
- **Google Tag Manager**: Manage and deploy website tags
- **Plausible**: Simple and privacy-friendly analytics

### Integration Methods

<Steps>
  <Step title="Access Website Settings">
    From your dashboard, select the website and navigate to Settings > SEO or
    Scripts.
  </Step>
  <Step title="Enter Tracking ID">
    Add your tracking ID or pixel code from the analytics platform.
  </Step>
  <Step title="Enable Enhanced Features">
    Configure any additional tracking features you need.
  </Step>
  <Step title="Verify Installation">
    Use the verification tools to ensure tracking is working correctly.
  </Step>
</Steps>
