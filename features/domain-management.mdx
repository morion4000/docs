## Overview

Manage domains at [app.getsite.ai/domains](https://app.getsite.ai/domains). Buy new domains, transfer existing ones, link external domains via CNAME, or get a free subdomain.

## Domain Options

| Option | Link | Description |
|--------|------|-------------|
| **Buy a domain** | [/domains/buy](https://app.getsite.ai/domains/buy) | Purchase and register a new domain through the platform |
| **Transfer a domain** | [/domains/transfer](https://app.getsite.ai/domains/transfer) | Transfer an existing domain to GetSite for full DNS control |
| **Link a domain** | [/domains/link](https://app.getsite.ai/domains/link) | Point an external domain to your site via CNAME |
| **Free subdomain** | [/domains/get](https://app.getsite.ai/domains/get) | Get a free subdomain on getsite.online or getsite.co |

### Domain Limits

| Plan | Custom Domains |
|------|---------------|
| Free | 0 |
| Maker | 10 |
| Developer | 50 |
| Max | Unlimited |

## Domain Settings

Each domain has a settings page ([app.getsite.ai/domains/:domainId](https://app.getsite.ai/domains/:domainId)) with:

| Tab | Description |
|-----|-------------|
| **Links** | Assign sites to paths on the domain |
| **Redirects** | URL redirect rules (301 redirects) |
| **Emails** | Email alias forwarding (transferred domains only) |
| **Analytics** | Domain-level traffic analytics |
| **Uptime** | Uptime monitoring with email alerts |
| **DNS** | DNS record management |
| **Settings** | General domain configuration |

## Email Aliases

For transferred domains, you can create email aliases (e.g., `hello@yourdomain.com`) that forward to any email address. Managed through Cloudflare Email Routing.

## Uptime Monitoring

Enable uptime monitoring on any domain to receive email alerts when your site goes down or recovers. Checks run every minute. Available on Maker plan and above.

## Usage Instructions

### Purchasing a New Domain

<Steps>
  <Step title="Access Domain Purchase">
    Navigate to
    [app.getsite.ai/domains/buy](https://app.getsite.ai/domains/buy).
  </Step>
  <Step title="Search for Domain">
    Enter your desired domain name and check availability.
  </Step>
  <Step title="Select Domain Options">
    Choose from available TLDs and review pricing.
  </Step>
  <Step title="Complete Purchase">
    Enter registration information and complete the payment process.
  </Step>
  <Step title="Link to Website">
    Select which website you want to connect to this domain.
  </Step>
</Steps>

### Transferring a Domain

<Steps>
  <Step title="Access Domain Transfer">
    Navigate to
    [app.getsite.ai/domains/transfer](https://app.getsite.ai/domains/transfer).
  </Step>
  <Step title="Enter Domain Information">
    Provide the domain name you wish to transfer.
  </Step>
  <Step title="Change NS Records">
    Change the NS records to the ones provided by GetSite.
  </Step>
</Steps>

### Linking an External Domain

<Steps>
  <Step title="Access Domain Linking">
    Navigate to
    [app.getsite.ai/domains/link](https://app.getsite.ai/domains/link).
  </Step>
  <Step title="Enter Domain Information">
    Provide the domain you wish to link.
  </Step>
  <Step title="Configure DNS">
    Follow the instructions to add the necessary CNAME record at your domain
    provider.
  </Step>
  <Step title="Link to Website">
    Select which website you want to connect to this domain.
  </Step>
</Steps>

### Getting a Free Subdomain

<Steps>
  <Step title="Access Free Subdomain">
    Navigate to
    [app.getsite.ai/domains/get](https://app.getsite.ai/domains/get).
  </Step>
  <Step title="Choose Subdomain Name">
    Enter your preferred subdomain prefix for your getsite.online or getsite.co domain.
  </Step>
  <Step title="Check Availability">
    Verify that your chosen subdomain is available.
  </Step>
  <Step title="Link to Website">
    Select which website you want to connect to this subdomain.
  </Step>
</Steps>
