## Overview

The GetSite AI Editor lets you modify your website with a combination of visual tools and AI assistance. Edit text, images, and sections directly, or use the AI Developer to make changes in plain language.

## Editor Features

### Visual Editing

Click on any element to open editing options:

- **Edit text, links, and images** — Click on any element to modify it directly.
- **Edit sections** — Click on any section to edit its code directly or use AI to regenerate it.

### AI Developer

Open the chat and tell the AI Developer what you want:

- **Add sections or change layout** — Describe what you want in plain language and the AI implements the changes.
- **Content refinement** — Ask the AI to improve copy, add features, or restructure content.

### Full Code Editor

Edit HTML, CSS, and JavaScript directly with syntax highlighting. Available on Developer plan and above.

### Real-time Preview

- **Live Updates**: See your changes immediately in the preview
- **Responsive Preview**: Test how your site looks on different device sizes

## Usage Instructions

<Steps>
  <Step title="Access the Editor">
    From your dashboard, select a website and click "Edit" or click on the edit
    icon next to any website.
  </Step>
  <Step title="Edit Content">
    Click on any text, link, or image to edit it directly. Click on a section to edit its code or regenerate it with AI.
  </Step>
  <Step title="Use the AI Developer">
    Open the chat to tell the AI what changes you want. It understands plain language and implements the changes.
  </Step>
  <Step title="Preview Changes">
    Use the preview pane to see how your changes will look on the live site.
  </Step>
  <Step title="Save and Publish">
    Click "Save" to store your changes (creates a new version) and "Publish" to make them live.
  </Step>
</Steps>

## Version History

Every save creates a new version with full HTML, CSS, and JavaScript:

- **Restore** any previous version with one click.
- **Fork** a version as a new site to experiment without affecting the original.
- **Export to CodePen** for external editing or collaboration.
- **Download as ZIP** to get a complete copy of your website files.
