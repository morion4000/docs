## Overview

GetSite's Asset Management feature allows you to upload, organize, and access your digital assets all in one place. This comprehensive solution ensures that your website images, files, AI-generated content, and other digital resources are well-organized and easily accessible when building and maintaining your websites.

## Key Features

### File Upload

Upload and manage files directly in the platform:

- **Drag-and-Drop**: Simple interface for uploading files
- **Bulk Upload**: Upload multiple files simultaneously
- **File Organization**: Create folders and organize your assets
- **File Preview**: Preview files before using them
- **Version Control**: Track different versions of your assets

### Asset Library

Organize and access your digital assets:

- **Filtering**: Filter assets by type, date, or custom tags
- **Search Functionality**: Quickly find specific assets
- **Metadata Management**: Add and edit asset metadata
- **Usage Tracking**: See where each asset is being used
- **Favorites**: Mark frequently used assets as favorites

### Image Library

Manage your images created with Flux (AI image generation):

- **Generated Images**: Access all AI-generated images
- **Regeneration**: Modify and regenerate images as needed
- **Image Editing**: Basic editing capabilities for images
- **Organization**: Group related images together
- **Download**: Export images for use outside of GetSite

### Website Designs

Manage your website designs created with Flux:

- **Design Templates**: Access all AI-generated website designs
- **Design Modification**: Edit and customize designs
- **Template Reuse**: Apply existing designs to new websites
- **Design Sharing**: Share designs with team members
- **Version History**: Track changes to designs over time

## Usage Instructions

<Steps>
  <Step title="Access Asset Management">
    Navigate to
    [app.getsite.ai/assets?tab=uploads](https://app.getsite.ai/assets?tab=uploads).
  </Step>
  <Step title="Upload Files">
    Click the "Upload" button or drag and drop files into the designated area.
  </Step>
  <Step title="Organize Files">

  </Step>
  <Step title="Edit Metadata">

  <Step title="Use in Websites">

  </Step>

## Asset Types

GetSite supports various types of digital assets:

- **Images**: JPG, PNG, GIF, SVG, WebP
- **Documents**: PDF, DOC, DOCX, TXT, XLS, XLSX
- **Media**: MP4, MP3, WebM
- **Code Files**: HTML, CSS, JavaScript, JSON
- **Fonts**: TTF, OTF, WOFF, WOFF2
- **AI-Generated Assets**: Images and website designs created with Flux

## Credits and Generation

Creating AI-generated assets requires credits:

- **Standard Image Generation**: 1 credit per image
- **Quality Image Generation**: 2 credits per image
- **Website Design Generation**: 10 credits per design

For more information about credits, see the [Credits System](/pricing/credits) documentation.

## Best Practices

- **Consistent Naming**: Use clear, descriptive names for all assets
- **Regular Organization**: Create and maintain a logical folder structure
- **Image Optimization**: Compress images before uploading to improve website performance
- **Metadata Usage**: Add detailed metadata to make assets more searchable
- **Version Control**: Keep track of different versions of important assets
- **Regular Backup**: Download important assets periodically as a backup
