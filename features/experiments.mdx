Run experiments to optimize your websites. Available on Max plan only.

Manage at [app.getsite.ai/experiments](https://app.getsite.ai/experiments).

## How it works

1. Create an experiment with multiple site variants.
2. Set traffic distribution weights.
3. Track conversion events.
4. The system detects the winner automatically.

Experiments go through four stages: **Draft** → **Running** → **Paused** → **Completed**.

You can also manage experiments from a site's **A/B Tests** tab in site settings.
