## Overview

Access utility tools at [app.getsite.ai/tools](https://app.getsite.ai/tools). These AI-powered tools help you generate design mockups, SEO metadata, logos, email copy, and domain name suggestions.

## Available Tools

| Tool | Description |
|------|-------------|
| **Design** | Generate a website design mockup from a prompt |
| **Metadata** | Generate SEO metadata for your site |
| **Logo** | Generate logo concepts |
| **Email** | Generate email copy |
| **Domain** | Get AI-powered domain name suggestions |

### Design Tool

Generate website design mockups from a text description. Costs 5 credits per design.

### Metadata Tool

Generate SEO-optimized page titles and meta descriptions for your websites.

### Logo Tool

Generate logo concepts with AI assistance.

### Email Tool

Generate email copy for your marketing campaigns. Costs 1 credit.

### Domain Tool

Get AI-powered domain name suggestions for your business. Costs 1 credit.

## Usage Instructions

<Steps>
  <Step title="Access Tools Dashboard">
    Navigate to [app.getsite.ai/tools](https://app.getsite.ai/tools) to see all
    available tools.
  </Step>
  <Step title="Select Tool">
    Click on the tool you want to use.
  </Step>
  <Step title="Provide Input">
    Describe what you need and let the AI generate results.
  </Step>
  <Step title="Use Results">
    Apply the generated output to your websites or campaigns.
  </Step>
</Steps>
