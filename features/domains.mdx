Manage domains at [app.getsite.ai/domains](https://app.getsite.ai/domains).

## Options

| Option | Description | Link |
|--------|-------------|------|
| **Buy** | Purchase a new domain | [/domains/buy](https://app.getsite.ai/domains/buy) |
| **Transfer** | Transfer a domain to GetSite for full DNS control | [/domains/transfer](https://app.getsite.ai/domains/transfer) |
| **Link** | Point an external domain via CNAME | [/domains/link](https://app.getsite.ai/domains/link) |
| **Free subdomain** | Get a subdomain on getsite.online or getsite.co | [/domains/get](https://app.getsite.ai/domains/get) |

## Domain Settings

Each domain has a settings page at `app.getsite.ai/domains/:domainId`:

| Tab | What it does |
|-----|-------------|
| **Links** | Assign sites to paths on the domain |
| **Redirects** | 301 redirect rules |
| **Emails** | Email alias forwarding (transferred domains only) |
| **Analytics** | Domain-level traffic stats |
| **Uptime** | Uptime monitoring with email alerts (checks every minute) |
| **DNS** | DNS record management |
| **Settings** | General configuration |

## Email Aliases

For transferred domains, create email aliases like `hello@yourdomain.com` that forward to any address. Managed through Cloudflare Email Routing.

## Uptime Monitoring

Get email alerts when your site goes down or recovers. Checks run every minute. Available on Maker plan and above.
