## Overview

GetSite's website creation feature lets you build fully designed, hosted websites from a text prompt in under 60 seconds. The AI generates complete HTML, CSS, and JavaScript with relevant copy, professional images, and responsive layouts.

## Website Creation

Create websites in three ways:

- **Prompt** ([app.getsite.ai](https://app.getsite.ai)) — Describe what you want and the AI generates a full site.
- **AI Chat** ([app.getsite.ai/chat](https://app.getsite.ai/chat)) — Have a conversation with the AI to build your site step by step.
- **Clone & Remix** — Enter any URL and the AI creates a modern, customizable version inspired by it.

Sites are automatically deployed to a free subdomain on `getsite.online`.

For detailed instructions, see the [How to Create a Website](/create-website) guide.

## AI Editor

The built-in [Website Editor](/features/website-editor) lets you modify your website with a combination of visual tools and AI assistance:

- **Edit text, links, and images** — Click on any element to open editing options.
- **Edit sections** — Click on any section to edit its code directly or use AI to regenerate it.
- **Add sections or change layout** — Open the chat and tell the AI Developer what you want. It understands plain language and implements the changes.
- **Full code editor** — Edit HTML, CSS, and JavaScript directly (Developer plan and above).
- **Save and publish** — Save your changes, then publish to make them live.

## AI Image Generation

- Generate custom AI images from text descriptions using Flux models.
- Standard quality (1 credit) and high quality (2 credits) options.
- All AI-generated images are royalty-free and commercially usable.
- Stock photo search via Unsplash is also available.

## Version History

Every save creates a new version with full HTML, CSS, and JavaScript:

- **Restore** any previous version with one click.
- **Fork** a version as a new site to experiment without affecting the original.
- **Export to CodePen** for external editing or collaboration.
- **Download as ZIP** to get a complete copy of your website files.

## Site Settings

Each site has a settings page ([app.getsite.ai/sites/:siteId](https://app.getsite.ai/sites/:siteId)) with the following tabs:

| Tab | Description |
|-----|-------------|
| **Links** | Assign custom domains and view live URLs |
| **SEO** | Page title, meta description, Open Graph image |
| **Scripts** | Inject custom HTML/CSS/JavaScript in the head or body |
| **Analytics** | Traffic analytics (visitors, page views, referrers) |
| **Forms** | View form submissions from the site |
| **Versions** | Version history and rollback |
| **A/B Tests** | Manage experiments assigned to the site |
| **Settings** | Site name, password protection, delete |

## SEO

- Custom page titles and meta descriptions.
- Open Graph tags for social sharing.
- Automatic XML sitemaps and robots.txt per domain.
- All sites are responsive by default (a key ranking factor).

## Custom Scripts

Add third-party scripts to extend your website:

- **Head section** — Analytics, tracking pixels, fonts.
- **Body end** — Chat widgets, marketing tools, payment buttons.
- Supports Google Analytics, Facebook Pixel, HotJar, Intercom, Crisp, Mailchimp, and any custom JavaScript.

## Dashboard

The [Dashboard](https://app.getsite.ai/dashboard) provides an overview of all your websites:

- **Website Listings**: View all your created websites in one place
- **Quick Actions**: Edit, publish, or manage settings for any site
