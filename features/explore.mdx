## Overview

Browse a directory of public websites created with GetSite. Discover community-created websites and clone any public site as a starting point for your own project.

Visit [app.getsite.ai/explore](https://app.getsite.ai/explore) to explore public sites.

## Key Features

### Discover Websites

- **Browse Directory**: Explore a collection of public websites created by the GetSite community
- **Find Inspiration**: See what others have built for ideas and inspiration

### Clone Sites

- **Clone & Remix**: Clone any public site as a starting point for your own project
- **Customize**: Use the AI editor to modify the cloned site to fit your needs

## Usage Instructions

<Steps>
  <Step title="Browse Public Sites">
    Navigate to [app.getsite.ai/explore](https://app.getsite.ai/explore) to browse the directory.
  </Step>
  <Step title="Find a Site">
    Browse or search for a site that matches what you're looking for.
  </Step>
  <Step title="Clone the Site">
    Click on a site to view it, then clone it as a starting point for your own project.
  </Step>
  <Step title="Customize">
    Use the [Website Editor](/features/website-editor) to customize the cloned site.
  </Step>
</Steps>
